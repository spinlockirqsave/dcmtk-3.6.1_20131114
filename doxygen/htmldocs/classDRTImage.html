Detailed Description

Helper class for working with radio therapy image objects.

This class provides a binding between DicomImage and DRTImageIOD. The information from the dataset are managed by the DRTImageIOD base class, but the image itself is represented through DicomImage.

Note
These functions hopefully simplify working with DRTImageIOD. This is a work in progress, please propose any improvements which would make working with this class easier for you and that your consider useful.
