Detailed Description

Accept or reject DcmItem objects by comparing a specific attribute to a set of values.

DcmAttributeFilter objects can be used to compare a specific attribute with one or more values - or to accept any DcmItem if no attribute is specified. DcmAttributeFilter objects can be used like any funtion taking a DcmItem& or a DcmFileFormat& object as parameter and returning a boolean value.
