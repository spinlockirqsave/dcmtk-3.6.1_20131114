Detailed Description

helper class for working with radio therapy plan objects.

This class provides several helper functions on top of DRTPlanIOD.

Note
These functions hopefully simplify working with DRTPlanIOD. This is a work in progress, please propose any improvements which would make working with this class easier for you and that your consider useful.
