Detailed Description

helper class for working with radio therapy structure set objects.

This class provides several helper functions on top of DRTStructureSet.

Note
These functions hopefully simplify working with DRTStructureSetIOD. This is a work in progress, please propose any improvements which would make working with this class easier for you and that your consider useful.
